## Step 2. Report it. 
- Add it to the list below. 
- Try to get the account banned.
- Send it to [Bellingcat](https://www.bellingcat.com/) - they are constantly debunking fakes, as well as [EUvsDisinfo](https://euvsdisinfo.eu/contact-us/).
